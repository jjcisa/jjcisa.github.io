现代网页布局

弹性布局

应用场景：小兔鲜项目（大部分单栏两端（between）对齐）、可以把行内元素转换成块级元素进行多种对齐方式

核心：1.父控子，父为容器，子为项目。2.主轴（排列方向）和交叉轴（移动方向）

特点：1.若子元素有大小，按照给定大小显示2.若子元素没有大小，拉伸充满父容器3.若子元素总宽度超过容器宽度，默认压缩子元素 。

对齐方式

主轴对齐方式

   justify-content: flex-start; 
   justify-content: flex-end; 
   justify-content: center;
   justify-content: space-between; 
   justify-content: space-around; 
   justify-content: space-evenly;

交叉轴对齐方式

  /* 交叉轴对齐方式：align-items单行有效 */
  align-items: flex-start;
  align-items: flex-end; 
  align-items: center;
  align-items: stretch; 
  /* 前提子盒子没有高度，但是好像没用，因为默认是拉伸充满父容器 */

主轴方向切换

 /* 定义主轴方向 */
  flex-direction: row;
  flex-direction: column; 
  /*主轴改变，交叉轴跟着变*/
  /* 注：行内元素有flex布局之后，就能够设置宽度和高度 */

换行

  /* 换行 */
  flex-wrap: wrap;
  /* 定义换行之后，align-content定义多行时交叉轴的对齐方式 */
  align-content: start;
  align-content:end;
  align-content: center;
  align-content: space-between;
  align-content: space-around;
  align-content: space-evenly;

间距

gap 控制子元素之间（盒子和盒子之间）的间距，实现了最后一个元素右间距去掉的效果

  gap: 20px;
  gap: 20px 30px;
  gap:20px 10px 30px;
  gap: 20px 10px 10px 5px;

弹性划分

  • flex 子盒子大小由父盒子剩余空间划分

  • flex可实现子盒子宽高由父盒子剩余空间灵活分配的效果

  • flex 1：是flex-grow(放大比例）1，flex-shrink（缩小比例）1，flex-basis（初始大小）的缩写0,注flex-basis优先级高于width和height

  /* flex 子盒子大小由父盒子剩余空间划分 */
  flex: 1;

定位布局

子绝父相

margin:0 auto对于定位的盒子无效，主要通过left,right，top,bottom这些方位词来确定

居中技巧：绝对定位：50%，margin-top：-xxpx

相对定位

特点：

1，不脱离正常流，元素原位置仍被保留

2.优先级：上》下，左》右

3.相对于 自身原来位置 偏移

display:relative;

绝对定位

特点：

1.脱离正常流，元素原位置被释放

2.优先级：上》下，左》右

3.相对于最近的已定位祖先元素(为了不影响布局，这个祖先元素往往使用相对偏移）偏移，如果没有已定位祖先元素，则相对于视口偏移

display:absolute;

固定定位

特点：

1.脱离正常流，不占用空间

2.上》下，左》右

3.始终相对于浏览器视口定位，滚动页面时位置不变

粘性定位

特点：

1.元素滚动到指定位置前是相对定位，之后转为固定定位

2.父容器overflow需为visible，否则粘性定位失效

3.需指定四个方位词中至少一个，否则无效

 position: sticky;
 top: 0;
/*这两个常配合使用*/

层叠顺序

1.值类型：正数负数，零，auto

2.默认值auto,后出现者覆盖前者

3.仅对定位元素有效

z-index:99;

常见组合：

固定导航栏：固定定位+z-index

弹窗：固定定位+居中技巧+高z-index

吸顶效果：粘性定位+top

总结

定位类型 脱离文档流 定位基准 层叠控制 典型场景
static 否 默认文档流 无 普通元素布局
relative 否 自身正常位置 支持 微调元素、子绝父相
absolute 是 最近定位祖先/视口 支持 弹窗、下拉菜单、悬浮元素
fixed 是 视口 支持 固定导航栏、返回顶部按钮
sticky 否 视口或滚动祖先 支持 吸顶导航栏、侧边栏固定

网格布局

创建网格轨道

行轨道和列轨道

 grid-template-columns: 200px 200px 200px;
  grid-template-rows: repeat(3, 1fr);

轨道对齐方式

 justify-content: space-between;
 align-content: space-between;

项目对齐方式

给单元格内的对齐方式，前提给子元素指定宽高,写到父元素上

  /* start,end,center,stretch(默认） */
  justify-items: center;
  align-items: center;

gap网格间隙

gap: 10px;
/*同弹性布局gap方式*/

###自动填充顺序

网格线

三列有4根列线，且网格线有固定编号

 /* 跨列 :写到子元素上，不要写到父亲上*/
  grid-column: 1 / 3;
  grid-row: 1 / 3;
  /* 第二种写法 */
  grid-column: 1/ span 2;
  grid-row: 1/ span 2;

网格填充顺序

  grid-auto-flow: row;
  /* 默认是按行填充 */
  grid-auto-flow: column;

object-fit

用于控制替换元素内容如何适应其容器尺寸的属性

  object-fit: cotain;
  object-fit: cover;
  /* 默认值 */
  object-fit: fill;

###响应式布局

当容器空间远远大于所有子元素大小：1.auto-fill留有空白空间。2.auto-fit会拉伸盒子占满父容器空间

grid-template-columns: repeat(auto-fill, minmax(160px, 1fr));

多列布局

特点：将元素的内容自动分割为指定数量垂直列

应用场景：瀑布流，长文章分栏等

  /* 直接在父盒子中写如下代码 */
  column-count: 3;
  column-gap: 20px;
  /* 列间线的样式 */
  column-rule: 1px solid #ccc;
  /* 强制元素不跨列切割：对子盒子写如下代码 */
  break-inside: avoid-column;
  /* 对于谷歌浏览器兼容 */
  -webkit-column-break-inside: avoid-column;
